About Choose UX

Choose UX is a living document, versioned in GitHub. ChooseUX/master should be considered the canonical branch. If you are going to make changes to the style guide, please make your changes in your own branch and use a pull request to get it into ChooseUX/master.

Designs and interactions in Choose UX are not always meant for copy and paste implementation into other products. They are meant to be a representation of what the resultant interface should look like or how it should function.

Questions or comments should be submitted on GitHub Issues.

Typography

Headlines for Choose Energy. Strong. Emphasis.

We use Franklin Gothic URW Demi for our headlines. The grotesque flourishes on Franklin Gothic like in the "g" and the cut of various descenders provides a distinct feel.

@include typeface-headline() or $font-family-headline + $font-weight-headline
Subheaders for Choose Energy. Strong. Emphasis.

Like our primary headlines, we use Franklin Gothic URW for our subheads; however, instead of the Demi weight, we use the Book weight for our subheads in order to provide appropriate contrast between our headlines.

@include typeface-subhead() or $font-family-subhead + $font-weight-subhead
Accent text for Choose Energy. Strong. Emphasis.

For accent copy like pull quotes and blockquotes, we use Merriweather Regular. It provides a nice contemporary serif that works well with the more modern feel of the body copy.

@include typeface-accent() or $font-family-accent + $font-weight-accent
Body copy text for Choose Energy. Strong. Duis mollis, est non commodo luctus, nisi erat porttitor ligula, eget lacinia odio sem nec elit. Emphasis. Etiam porta sem malesuada magna mollis euismod. All the things.

For body copy and user interface text, we use Proxima Nova Light, a modern geometric typeface with limited flourishes to contrast with the headlines.

@include typeface-body() or $font-family-body + $font-weight-body
