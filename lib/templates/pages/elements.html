Elements

Our matter enables us to be adaptive.

element (n.) — a small atomic unit of interface that is used to build larger components

What are Elements?

atomic

  • Smallest unit of user interface in our pattern library
  • Cannot be broken down more without losing its meaning

reference

  • Useful for seeing our global styles in use at once.
  • Quick at-a-glance reference for a given element's base styles.

lightweight

  • Styles of elements should be minimal to create a clean slate
  • Typically requires combining with other elements and styles to build a component
