Content Structures

Editorial Block

The editorial block is a bit unique in its construction. There is a standard .content-editorial component, but it generally will require a context to be useful. Each of the editorial blocks below has a different context, listed beforehand. Typically, these will be art-directed and delivered as completed HTML and CSS for implementation.

It should also be noted that every editorial block has content set to the $site-max-width but expands well beyond that at 100% width for its background image.
