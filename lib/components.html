Footer

The Footer should be included in all projects either as an Angular.JS Component or a standard HTML block. By default, all projects should be the light footer style. Each footer is comprised of a Primary and Secondary Footer component with multiple Footer Nav Boxes in the Primary Footer and three distinct components in the Secondary Footer.

Primary Footer

The Primary Footer is a lighter section of the footer designed to be the section of the footer with which the user typically interacts. It should contain all of the links to other sections of the product as well as any SEO-relevant content (i.e. recent blog posts, popular searches, links to crawl-ready landing pages).

Footer Nav Boxes

A footer nav box (fnav-box) is a component used to encapsulate a header and nav-list containing an arbitrary number of nav-link elements. There are two variants: standard and wider. The fnav-wider modifier can be added to a given fnav-box to produce a double-width column.

Secondary Footer

The Secondary Footer is a darker section of the footer designed to be populated with social and legal content that is less important for a user to interact with from this location but is often useful to access on any page of the site.
